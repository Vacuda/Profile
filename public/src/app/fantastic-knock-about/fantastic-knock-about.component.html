A Fantastic Knock About - Game Jam Game

The theme of this game jam was "An End Is A New Beginning." I wanted to explore what happens after you lose in one of those classic atari games. It's a fun little concept, having paddles now on each side of the screen. There are some procedurally-generated aspects to it which makes it different each time.
